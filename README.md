# Personal Website

Mencipta karya melalui rekayasa perangkat lunak.

This repository contains the source code for my personal website — a living space where portfolio, writing, and ongoing learning converge. It is both a technical playground and a structured archive of how my thinking evolves over time.

## Overview

The website functions as:

- A portfolio of selected works and experiments
- A blog for long-form reflections on technology, culture, and systems
- A public index of skills, tools, and ongoing exploration

It is designed not merely as a static showcase, but as an evolving system.
